Descrição do Projeto

Desenvolvimento de um sistema completo de gestão empresarial (ERP) para controle de estoque, vendas, financeiro e recursos humanos. O sistema deve ser web-based, responsivo e integrado com APIs de bancos e sistemas de pagamento.

Principais funcionalidades incluem: dashboard executivo, controle de inventário, gestão de clientes e fornecedores, módulo financeiro completo, relatórios avançados e sistema de usuários com diferentes níveis de acesso.
